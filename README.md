# Object Detection-based Grocery POS system

### Features:
- Hand Gesture Recognition for starting or stopping object detection inference.
- YOLOv8n Object Detection Model trained on custom grocery dataset of 33 objects/classes.
- compiled with ONNX or TensorRT for real-time 60FPS inference.
